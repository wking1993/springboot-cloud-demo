1.Zuul是Netflix OSS的一员，是一个基于JVM路由和服务端的负载均衡器；提供路由、监控、弹性、安全等方面的服务框架；
  Zuul的核心是过滤器，通过这些过滤器，可以扩展许多功能：动态路由、请求监控、认证鉴权、压力测试、灰度发布。
2.Zuul中的过滤器分为4中：
  1）pre：在请求被路由之前调用。适用于身份认证的场景；
  2）route：在路由请求时被调用。适用于灰度发布场景；
  3）post：在route和error过滤器之后被调用。适用于添加响应头，记录响应日志等场景；
  4）error：处理请求发生错误时调用。可以用来统一记录错误信息；
  *可以通过ZuulServlet源码的service()方法来了解整个执行顺序。
